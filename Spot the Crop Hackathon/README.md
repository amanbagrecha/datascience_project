# Radiant Earth Spot the Crop Hackathon by #ZindiWeekendz

Based on time-series of Sentinel-1 satellite images to classify crop types in South Africa.

The evaluation metric for this challenge is Cross Entropy with binary outcome for each crop. Competition closed on 4 July 2021.

The [notebook](https://github.com/amanbagrecha/datascience_project/blob/master/Spot%20the%20Crop%20Hackathon/ZINDI_SPOT_THE_CROP_CHALLENGE.ipynb) contains solution to get **5th position** on public leaderboard!
